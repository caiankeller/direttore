# Direttore

Workspace for the Direttore package family.

## Structure

```txt
packages/
  core/         @direttore/core
  react-query/  @direttore/react-query
```

Shared development config lives at the workspace root. Package-specific source,
tests, examples, README, and publish metadata live inside each package folder.

## Documentations

[`@direttore/core`](/packages/core): zero-dependency API orchestration primitives.

[`@direttore/react-query`](/packages/react-query): TanStack React Query adapter for mounted endpoints.
